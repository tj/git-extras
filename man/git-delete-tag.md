git-delete-tag(1) -- Delete tags
================================

## SYNOPSIS

`git-delete-tag` &lt;tagname&gt;

## DESCRIPTION

  Deletes local and remote tag named &lt;tagname&gt;.

## OPTIONS

  &lt;tagname&gt;

  The name of the branch to delete.

## EXAMPLES

    $ git delete-tag 0.0.1
