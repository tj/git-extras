NAME

git-bug - Create bug branch

SYNOPSIS

git-bug [finish] <name>

DESCRIPTION

Create the given bug branch

OPTIONS

<finish>

Merge and delete the bug branch.

<name>

The name of the bug branch.

EXAMPLES

$ git bug bug-123456
...
$ git commit -m "Some changes"
...
$ git checkout master
$ git bug finish bug-123456
