git-touch(1) -- Touch and add file to the index
===============================================

## SYNOPSIS

`git-touch` &lt;filename&gt;

## DESCRIPTION

  Call `touch` on the given file and add it to the current index. Used one-step creating new files.

## OPTIONS

  &lt;filename&gt;

  File to be touched.

## EXAMPLES

  $ git touch Makefile
