git-graft(1) -- Merge and destroy a given branch
======================================

## SYNOPSIS

`git-graft` &lt;src-branch&gt; &lt;dest-branch&gt;

## DESCRIPTION

  Merge commits from &lt;src-branch&gt; into &lt;dest-branch&gt;

## OPTIONS

  &lt;src-branch&gt;

  &lt;dest-branch&gt;

## EXAMPLES

    $ git graft new_feature dev
