git-delete-submodule(1) -- Delete submodules
============================================

## SYNOPSIS

`git-delete-submodule` &lt;path&gt;

## DESCRIPTION

## OPTIONS

  &lt;path&gt;

  The path of the submodule to delete.

## EXAMPLES

    $ git delete-submodule lib/foo
