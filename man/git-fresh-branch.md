git-fresh-branch(1) -- Create fresh branches
============================================

## SYNOPSIS

`git-fresh-branch` &lt;branchname&gt;

## DESCRIPTION

  Creates empty local branch named &lt;branchname&gt;.

## OPTIONS

  &lt;branchname&gt;

  The name of the branch to create.

## EXAMPLES

    $ git fresh-branch docs
