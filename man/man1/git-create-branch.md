git-create-branch(1) -- Create branches
=======================================

## SYNOPSIS

`git-create-branch` &lt;branchname&gt;

## DESCRIPTION

  Creates local and remote branch named &lt;branchname&gt;.

## OPTIONS

  &lt;branchname&gt;

  The name of the branch to create.

## EXAMPLES

    $ git create-branch integration
