git-unlock(1) -- Unlock a file excluded from version control
============================================================

## SYNOPSIS

`git-lock` &lt;filename&gt;

## DESCRIPTION

  Unlock local files from version control.

## OPTIONS

  &lt;filename&gt;

  The name of the filename.

## EXAMPLES

    $ git unlock config/database.yml
