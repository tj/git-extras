git-lock(1) -- Lock a file excluded from version control
========================================================

## SYNOPSIS

`git-lock` &lt;filename&gt;

## DESCRIPTION

  Lock local files from version control.

## OPTIONS

  &lt;filename&gt;

  The name of the filename.

## EXAMPLES

    $ git lock config/database.yml
