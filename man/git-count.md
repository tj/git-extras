git-count(1) -- Show commit count
=================================

## SYNOPSIS

`git-count` [--all]

## DESCRIPTION

  Show commit count.

## OPTIONS

  --all

  Show commit count details.

## EXAMPLES


 Output commit total:

    $ git count

    total 1844
