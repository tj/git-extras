git-continue(1) -- Continue current git operation
================================

## SYNOPSIS

`git-continue`

## DESCRIPTION

  Continue current git revert, rebase, merge or cherry-pick process.

## OPTIONS

  There are no options, it just continues current operation.

## EXAMPLES

  `git-continue`
